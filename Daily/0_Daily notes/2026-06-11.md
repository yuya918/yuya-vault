## Pricing Insight
Presenzaの価格帯は、以下を中心に考える。

### Purchase Price
- Ideal: 20,000〜30,000円
- Acceptable: 30,000〜40,000円
- Careful: 40,000〜50,000円

### Selling Price
- Standard: 50,000〜60,000円
- Strong Item: 70,000〜80,000円
- Premium Test: 100,000円前後は慎重

粗利だけで判断せず、撮影素材として使えるか、SNSで世界観を作れるか、自分で着て説得力があるかも評価対象にする。

## Subtitle Automation
Claude Code / Fableが6/22まで使えるため、旅動画の字幕作成ワークフローを整理した。

目的は、海外旅行YouTubeロング動画の字幕作成を、Premiere Pro + Claudeで半自動化すること。

### Workflow
- 編集確定
- スピーチを強調
- Premiereで自動文字起こし
- 他人の発話を削除
- SRTを書き出し
- ClaudeにSRTを渡して整形
- 整形済みSRTをPremiereに再読み込み
- 【要確認】箇所だけ目視チェック
- 書き出し

この流れにより、字幕作成の負担を大きく減らせる可能性がある。

## Markdown Operation Rule
Obsidianに貼るMarkdownの形式についても運用ルールが明確になった。

今後は、Obsidian用のMarkdownは以下で統一する。

- 貼る本文は1つのMarkdownコードブロックだけ
- 補足説明と貼る本文を混ぜない
- コードブロック内はすべてObsidianに貼る内容
- 入れ子のコードブロックは使わない
- 空行を増やしすぎない
- `---` は原則使わない
- 数式や計算式も普通の文章として書く

このルールにより、Obsidianへの転記ミスや表示崩れを防ぐ。
