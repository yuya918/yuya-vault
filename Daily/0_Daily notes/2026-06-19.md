## Theme
疲労の自覚と、本業への集中日

## Summary
金曜日。  
今週は仕事の疲労がかなり強く、朝の時点で「仕事で疲れすぎている」「全然いけていない」「やる気が出ない」という感覚があった。

ただし、重要なのは、疲れていても毎日出社していたこと。  
やりきれていないタスクはあるが、やらなかったことより、やったことを見るべき日だった。

この日は、脳のキャパが追いついていない状態を自覚しながらも、「一個一個やるしかない」と整理し、本業にかなり集中する方針を立てた。

## 今日の位置づけ
6/19は、インプットや制作を広げる日ではなく、仕事の疲労を受け止めて、本業に集中する日。

6/17は実務インサイトの日。  
6/18は疲労と生活維持の日。  
6/19は、その疲労がまだ残る中で、改めて本業に向き合う日。

今週全体として、仕事量が多く、夜の余力やインプットの時間がかなり削られていた。  
ただ、その中でも出社し続け、最低限の責任を果たしていた。

## Work / 仕事の疲労
朝の時点で、仕事への疲労感がかなり強かった。

- 仕事で疲れすぎている
- 今週は全然いけていない感覚がある
- しんどすぎる
- 終わらない
- やる気が出ない
- タイピングする気持ちにもなれない

金曜日になると、週の疲労が蓄積していることをかなり実感した。  
これは気合いの問題ではなく、明確に脳と身体のリソースが削られている状態。

## それでも毎日出社した
重要なのは、疲れていても毎日出社したこと。  
完璧に仕事を終わらせられたわけではない。  
やりきれていないタスクもある。

それでも、今週は毎日会社に行き、仕事に向き合っていた。  
これは小さく見えるが、かなり大きい。

やらなかったことだけを見ると、自分を責めてしまう。  
でも、やったことを見ると、確実に踏みとどまっている。

## Task / 一個一個やるしかない
本業のタスクはかなり多い。  
脳のキャパが追いついていない感覚もある。

ただ、こういう時こそ、全部を一気に考えない。  
目の前のものを一つずつ処理するしかない。

今の状態で必要なのは、完璧な計画ではなく、順番に処理すること。

## 今日の方針
今日は本業にかなり集中する。  
TravelやPresenza、Obsidian、読書なども大事だが、今日はまず仕事。

今は本業の負荷が強く、そこから逃げずに処理していく必要がある。  
ただし、無理に全てを今日中に終わらせようとするのではなく、一個一個進める。

## Input / インプット状況
6/19も、知的インプット量としては多くない。  
仕事疲労が強く、読書や制作に深く入る余力はあまりなかった。

ただし、この日は「インプットできなかった日」として見るより、仕事の負荷を受け止めた日として記録する方が正しい。

6/17〜6/19は、外部インプットよりも実務・疲労・生活維持の比重が高い期間だった。

## Mental / 気持ち
疲労感が強く、自分に対して「全然いけていない」と感じる日だった。  
しかし、その感覚の中でも「一個一個やるしかない」と言語化できたことは大きい。

気持ちが落ちている時は、抽象的に自分を責めがちになる。  
でも実際には、毎日出社しているし、仕事にも向き合っている。

今は、自己否定よりもタスクの分解が必要。
