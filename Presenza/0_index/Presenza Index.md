## Overview
Presenzaは、イタリア中古セットアップを通じて、静かな上質さと生活の余白を提案するプロジェクト。

単なる中古販売ではなく、商品選定・サイズ提案・撮影・スタイリング・世界観・日本の都市生活への翻訳によって価値を作る。

## Current Phase
現在は、2着のテスト購入を通して商品仮説を検証している段階。

- Tagliatore 42
- Lardini

この2着を使って、サイズ感・状態・撮影適性・SNSでの見え方・販売価格・粗利を確認する。

## Core Price Range
### Purchase Price
- Ideal: 20,000〜30,000円

- Careful: 40,000〜50,000円

### Selling Price
- Standard: 50,000〜60,000円
- Strong Item: 70,000〜80,000円
- Premium Test: 100,000円前後は慎重

## Current Learning
Tagliatore 42は、約30,000円で購入でき、状態も想定より良かった。サイズ42も条件次第で使える可能性がある。

Lardiniは49,000円から45,000円に交渉して成立した。価格交渉によって粗利余地を作れることが分かった。

メルカリ上で競合が1名のみだったことから、特定ブランド・特定サイズ帯にはブルーオーシャンの可能性がある。
