# Target Customer

## Purpose
Presenzaが誰に向けたブランドなのかを明確にする。

商品選定、価格、撮影、Instagram運用、商品説明のすべてを、ターゲット顧客から逆算する。

## Core Target
Presenzaの主なターゲットは、20代後半〜30代中盤の男性。

年収はおよそ800〜1,500万円を想定する。

仕事にも生活にも一定の余裕が出始めているが、まだ高級ブランドを自由に買い続けるほどではない層。

## Profile
- Age: 28〜35歳
- Gender: 男性
- Income: 800〜1,500万円
- Height: 165〜170cm前後
- Body Type: 細身〜標準体型
- Location: 東京・神奈川・都市部
- Work Style: 外資IT、コンサル、金融、広告、スタートアップ、専門職
- Lifestyle: 仕事、カフェ、街歩き、旅行、食事、文化的な体験に関心がある

## Fashion Problem
既製品のスーツやジャケットに対して、以下のような違和感を持っている。

- サイズが大きすぎる
- 肩が強すぎる
- ビジネス感が強すぎる
- 私服として使いにくい
- 高級ブランドは高すぎる
- 古着は選び方が分からない
- 自分に合うサイズや色が分からない
- 上品に見せたいが、派手にはなりたくない

## Emotional Needs
Presenzaの顧客は、単に服が欲しいわけではない。

本当に欲しいのは、以下のような感覚。

- 静かに整って見えること
- 無理をしていないのに上品であること
- 都市の中で浮かないこと
- 仕事にも休日にも使えること
- 自分の生活に余白があるように見えること
- 派手ではないが、記憶に残ること
- 年齢に合った落ち着きがあること

## Price Psychology
ターゲット顧客の価格感覚は以下。

- 30,000円：即決しやすい
- 50,000円：悩むが購入可能
- 70,000〜80,000円：商品力と世界観があれば検討可能
- 100,000円：かなり慎重。初期フェーズでは狙いすぎない

Presenzaの基本販売価格は、50,000〜60,000円を中心に設計する。

強い商品だけ、70,000〜80,000円を検証する。

## Why Used Italian Setup
ターゲット顧客にとって、イタリア中古セットアップは合理的な選択肢になり得る。

理由は以下。

- 新品より価格が現実的
- 高品質な素材や仕立てを手に取りやすい
- 日本の量販スーツより雰囲気が出る
- ジャケット単体でも使える
- サイズが合えば一気に印象が整う
- 大量消費ではなく、選び抜く感覚がある

## Customer's Desired Image
顧客がなりたい姿は、以下。

- 仕事ができそうに見える
- 清潔感がある
- 余裕がある
- 文化的に見える
- 旅や都市が似合う
- 派手ではないが品がある
- 年齢より少し成熟して見える
- 高級ブランドを見せつけない

## Customer's Current Alternatives
ターゲット顧客は、現在以下の選択肢を持っている。

- ユニクロ
- Suit Select
- Global Style
- Tomorrowland
- United Arrows
- Beams
- Zegna / Lardini / Tagliatoreなどの新品
- メルカリや古着屋での自己判断購入

Presenzaは、これらの中間に位置する。

新品高級ブランドほど高くないが、量販店よりも世界観がある。

古着だが、ただの中古ではなく、選定・撮影・説明・サイズ提案によって価値を再編集する。

## What Presenza Offers
Presenzaが提供する価値は以下。

- 小柄〜標準体型に合いやすいサイズ提案
- イタリアブランドの上質さ
- 日本の都市生活に馴染む色とシルエット
- 静かな上品さ
- 撮影による着用イメージ
- 商品背景やスタイリング提案
- 価格と品質のバランス

## Customer Message
Presenzaが顧客に伝えるべきメッセージ。

高級感を見せつける必要はない。

自分に合う服を、静かに選ぶ。

少ない服で、生活はもっと整う。

## Avoided Customer
Presenzaが初期フェーズで無理に狙わない顧客。

- とにかく安さだけを求める人
- ブランドロゴを強く見せたい人
- 派手なファッションを求める人
- 10万円以上の服を迷わず買う富裕層
- サイズや素材よりも流行を優先する人
- 大量に服を買い替える人

## Current Hypothesis
Presenzaの初期ターゲットは、身長165〜170cm前後で、落ち着いた上質さを求める都市部の男性。

彼らは、派手なラグジュアリーではなく、静かに整った印象を求めている。

Presenzaは、その人たちに向けて、イタリア中古セットアップを日本の都市生活に合う形で再提案する。
