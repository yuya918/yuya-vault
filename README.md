# Yuya Vault

Yuya の Obsidian Vault です。

Daily / Travel / Presenza を軸に、日々の記録、週次・月次の振り返り、旅コンテンツ制作、Presenza の事業・制作管理をまとめます。

## Structure

```text
Daily/
├── 0_Daily notes/
├── 1_Weekly Index/
├── 2_Monthly Index/
└── 3_Yearly index/

Travel/
├── 0_index/
├── 1_Scripts/
├── 2_Titles/
├── 3_Production Logs/
├── 4_Country notes/
├── 5_Long videos/
├── 6_Shorts/
└── 7_Subtitle Automation/

Presenza/
├── 0_index/
├── 1_Philosophy/
├── 2_Strategy/
├── 3_Product/
├── 4_Content & SNS/
└── 5_Operations/
```

## Daily

日々の記録を `Daily/0_Daily notes/` に保存します。

命名形式:

```text
YYYY-MM-DD.md
```

週次・月次・年次の振り返りは、それぞれ以下にまとめます。

- `Daily/1_Weekly Index/`
- `Daily/2_Monthly Index/`
- `Daily/3_Yearly index/`

## Travel

Travel は旅・映像制作・SNSコンテンツの制作管理領域です。

- `0_index/`: Travel 全体の入口
- `1_Scripts/`: 動画やShortsの台本
- `2_Titles/`: タイトル案、命名ルール
- `3_Production Logs/`: 制作ログ
- `4_Country notes/`: 国・地域ごとのメモ
- `5_Long videos/`: 長尺動画関連
- `6_Shorts/`: Shorts案
- `7_Subtitle Automation/`: 字幕・自動化ワークフロー

## Presenza

Presenza はブランド・事業・制作運営の管理領域です。

- `0_index/`: Presenza 全体の入口
- `1_Philosophy/`: ブランド思想、コンセプト
- `2_Strategy/`: 事業戦略、価格、ターゲット
- `3_Product/`: 商品基準、仕入れ、テスト購入
- `4_Content & SNS/`: Instagramや撮影計画
- `5_Operations/`: 利益計算、出品テンプレート、運用

## GitHub Sync

このVaultは GitHub の `yuya918/yuya-vault` と同期します。

Obsidian Git を使う場合:

- Vault root: `/Users/maedayuuya/Desktop/Yuya`
- Remote: `https://github.com/yuya918/yuya-vault.git`
- Branch: `main`

推奨運用:

- Obsidianで編集する
- Obsidian Gitで commit / pull / push する
- GitHubをバックアップと履歴管理に使う

## Ignore Policy

以下はGitHubに載せません。

- `.DS_Store`
- `.obsidian/workspace.json`
- `.obsidian/workspace-mobile.json`
- Obsidian Git のローカル認証補助ファイル
