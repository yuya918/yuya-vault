# Chongqing Short

## Project
Chongqing / China Short

## Theme
北京・上海・深圳のような有名都市のイメージだけでは見えてこない、中国西部の巨大都市・重慶の「むき出しの熱量」。

## Progress
- 初期案では「中国の発展」「中国の目覚ましい発展」に寄りすぎていた。
- その後、過去ショートの型を参照しすぎて、ベトナム型の「一見バラバラに見えるけど共通点があった」に近づきすぎた。
- 議論を通じて、重慶のテーマは「共通点の発見」ではなく、「代表都市のイメージからはみ出す都市の迫力」だと整理。
- 最終的に、核心を「むき出しの熱量」に設定。
- 締めは「国の輪郭は、有名な都市だけでは見えてこない」に決定。
- サムネは「大国の熱量」を最有力に設定。

## Decisions
### Script
完成台本は [[Chongqing Short]] に格納。

### Title
最終候補は「大国の熱量」。  
英語サブコピーは「CHINA'S RAW ENERGY」。

### Thumbnail
黒背景・明朝体・赤1ワードの既存シリーズ感を維持。  
赤字は「熱量」。

### Core Line
街を歩けば歩くほど、ここには、むき出しの熱量があると感じました。

### Closing Line
国の輪郭は、有名な都市だけでは見えてこない。  
だから旅は、深くて面白い。

## Realization
既存の成功構成は参考にするが、そのままテンプレート化してはいけない。

ベトナム、サウジ、シンガポール、モンゴル、ダラヴィにはそれぞれの認識変化がある。  
重慶には重慶の認識変化が必要だった。

重慶の場合、テーマは「未来」「発展」ではなく、未来・産業・生活・歴史・人の熱気が混ざった「むき出しの熱量」。

## What Worked
- 冒頭を「中国って、北京、上海、深圳のような、巨大都市のイメージがありました。」に絞った。
- 重慶を「中国西部の経済を支える、巨大な製造業都市」と一文で説明した。
- 経済説明を数字ではなく、街の体感につなげた。
- 都市構造パートを短くした。
- 洪崖洞は「ホンヤードン」とふりがなを入れた。
- 吊脚楼の専門語を避け、「山の斜面に張りつくような伝統建築」とした。
- 「むき出しの熱量」を結論として後半に置いた。
- 最後を「国の輪郭は、有名な都市だけでは見えてこない」にした。

## What Did Not Work
- 「中国の発展」「中国の未来」と言い切ると浅くなる。
- 「中国の本気」は強いが、やや大げさで説明っぽい。
- 「一つのイメージだけでは説明できない」はダラヴィと被る。
- 「一見バラバラに見えるけど共通点があった」はベトナムと被る。
- 「最初に圧倒されたのは都市の形」は地形が主役になりすぎる。
- 「巴渝の伝統建築、吊脚楼を再現した」はナレーションとして硬い。
- 産業データを入れすぎると旅行解説動画になる。

## Next
- 映像を台本順に整理する。
- サムネ候補「大国の熱量」でラフ作成。
- 1分以内に収まるように読み上げ確認。
- 洪崖洞・ネオン・人混み・火鍋・立体都市の映像を優先的に配置。
- 英語字幕版を作る場合は [[Chongqing Short]] の英語字幕をベースにする。
