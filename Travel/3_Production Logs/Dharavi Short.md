# Dharavi Short

## Project
Dharavi / Mumbai / India Short

## Theme
スラムという一語では説明できない、都市の裏側を支える経済と人の力。

## Progress
- 過去の議論では、Dharaviの最強テーマは「スラム×経済」と整理。
- 単なる貧困紹介や危険な場所紹介ではなく、都市の裏側にある労働・生産・生活を見る方向に設定。
- タイトルは「スラムの経済」が最も強い。
- 締めは「街の奥には、数字では見えない経済がある。だから旅は、深くて面白い。」を軸にする。

## Decisions
### Script
完成台本は [[Dharavi Short]] に格納。

### Title
最終候補は「スラムの経済」。  
英語サブコピーは「MORE THAN A SLUM」。

### Thumbnail
黒背景・明朝体・赤1ワードの既存シリーズ感を維持。  
赤字は「経済」。

### Core Line
ダラヴィで見たのは、貧困だけではありませんでした。

### Closing Line
街の奥には、数字では見えない経済がある。  
だから旅は、深くて面白い。

## Realization
Dharaviは、「かわいそう」「危険」「貧困」という単純な語りにしてはいけない。

同時に、貧困や厳しい生活環境を無視して「活気がある」と美化してもいけない。

重要なのは、厳しい環境の中にある生活・労働・生産・小さな経済を見ること。

## What Worked
- 「世界最大級のスラム」という既存イメージから入る。
- 実際に歩いた認識変化として「そのイメージだけでは説明できない場所だった」と展開する。
- 路地・工房・ものを作る音を具体映像として使う。
- 革製品、陶器、リサイクル、縫製、食品加工を並べて都市経済を見せる。
- 「もちろん、厳しい生活環境があることは事実です」と入れて、美化を避ける。
- 最後に「街の奥には、数字では見えない経済がある」と知的に締める。

## What Did Not Work
- 危険・貧困・衝撃に寄せると消費的になる。
- 「インドの闇」系はブランドに合わない。
- 観光地紹介ではなく、社会観察として作る必要がある。
- 感動ポルノにしない。
- 貧困をロマン化しない。
- 産業説明を入れすぎると硬くなる。

## Next
- 実際の映像素材を確認する。
- 工房・路地・人の手元・生活感のある映像を優先的に使う。
- 1分以内に収まるように台本を削る。
- サムネ「スラムの経済」でラフ作成。
- 英語字幕は [[Dharavi Short]] のLine-by-Line Subtitle Pairをベースにする。
